# Nikhilesh Chinta — Portfolio

Personal portfolio website built with **React**, **TypeScript**, **Tailwind CSS** (v4), and **Vite**.

## Sections

- **Hero** — Animated typing effect, gradient branding, CTAs
- **About** — Professional summary with key stats
- **Experience** — Timeline of roles (Wipro, Astin IT Solutions, CloudyPeople)
- **Projects** — 5 featured project cards with highlights
- **Skills** — 6 categorized skill bars with percentage ratings
- **Contact** — Email, LinkedIn, GitHub links

## Development

```bash
npm install
npm run dev     # dev server at localhost:5173
npm run build   # production build to dist/
npm run preview # preview production build
```

## Tech Stack

React 19 · TypeScript · Tailwind CSS v4 · Vite · Inter Font
